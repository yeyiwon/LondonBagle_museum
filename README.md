## 🎨 주색
    #FFF5E0, #C70039, #141E46

### 선택 이유 

    가게의 색상은 고풍스러운 감성을 살리기 위해
    따뜻한 #FFF5E0 크림색을 중간에 섞어
    아늑하고 편안한 분위기를 주기위해, 

    영국 국기가 들어가있는 컨셉에 맞춰  #C70039  #141E46 색상으로 고급스러운 영국 빵집에 온 느낌을 주려고 사용
